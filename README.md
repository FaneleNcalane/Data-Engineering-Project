# Tokyo 2020 Olympics - Coaches and Teams Analysis

This repository contains comprehensive datasets from the Tokyo 2020 Olympics (held in 2021), focusing on coaches, athletes, teams, medals, and gender participation across various disciplines.

##  Dataset Overview

The repository includes five Excel files with detailed Olympic information:

### 1. **Coaches.xlsx**
Contains information about coaches participating in the Tokyo 2020 Olympics:
- Coach names
- National Olympic Committee (NOC) representation
- Sports disciplines
- Event categories (Men/Women/Mixed)

### 2. **Athletes.xlsx**
Comprehensive list of athletes who participated:
- Athlete names
- Country representation (NOC)
- Sports disciplines

### 3. **Teams.xlsx**
Team participation data:
- Team names
- Disciplines
- Representing countries
- Event types

### 4. **EntriesGender.xlsx**
Gender participation statistics by discipline:
- Number of female athletes
- Number of male athletes
- Total participants per discipline

### 5. **Medals.xlsx**
Complete medal tally:
- Country rankings
- Gold, Silver, Bronze counts
- Total medals
- Rank by total medals

##  Key Features

- **Comprehensive Coverage**: Data across all Olympic disciplines
- **Multi-dimensional Analysis**: Coach assignments, team compositions, gender participation, and medal performance
- **Global Representation**: Data from countries worldwide
- **Structured Format**: Easy to import into data analysis tools (Python, R, Excel)

##  Potential Analysis Areas

- Gender parity across different sports
- Coach nationalities vs. team performances
- Medal distribution patterns
- Team compositions by country
- Correlation between coach experience and team success
- Country-wise participation analysis
